# Network Monitoring Dashboard

This project monitors network devices and shows their status (UP/DOWN) using Python and Flask.

## Features

* Ping devices
* Show status in browser
* Simple dashboard

## Tech Stack

* Python
* Flask
* Linux (WSL)

## How to Run

```bash
pip install flask
python3 app.py
```

## Example Devices

* 127.0.0.1 (Localhost)
* 8.8.8.8 (Google DNS)
